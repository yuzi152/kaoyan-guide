# 调剂指导

## 适用场景
初试成绩公布后，分数未达到目标院校复试线，或擦线进复试但风险高，或复试被刷，需要走调剂通道。**调剂不是失败——每年约 30% 的研究生是通过调剂上岸的。**

---

## 一、调剂基本条件

### 硬性条件（必须满足）

1. **初试成绩过国家线**（A 区或 B 区）
   - 过 A 区线 → 可在 A 区和 B 区调剂
   - 仅过 B 区线 → 只能在 B 区调剂
   - 未过 B 区线 → 无法调剂（只能二战或工作）

2. **调入专业与第一志愿相同或相近**
   - 原则上在同一学科门类内（专业代码前两位相同）
   - 跨学科调剂需要调入专业有名额 + 接收跨学科考生

3. **初试科目与调入专业初试科目相同或相近**
   - 考 408 的可以调到考 408 的（最顺畅）
   - 考 408 的调到考自命题的 → 需要确认是否接受
   - 数学一可调到数学二/三，但反向不行

4. **符合调入院校的调剂要求**
   - 部分院校对本科层次有要求（如"985/211 优先"）
   - 部分专业有额外限制（如工作经验、特定背景）

---

## 二、调剂决策框架

在开始调剂前，先回答以下问题：

```
决策树：

初试总分是否过国家线？
├── 是 → 继续
│   ├── 总分比目标院校复试线低多少？
│   │   ├── < 5 分 → 等复试线公布，有可能擦线进复试
│   │   ├── 5-15 分 → 同步准备：目标院校复试 + 调剂
│   │   └── > 15 分 → 全力调剂，目标院校基本无望
│   └── 总分比 B 区线高多少？
│       ├── > 30 分 → 在 A 区调剂有竞争力
│       ├── 10-30 分 → A 区可调，B 区更稳妥
│       └── < 10 分 → 以 B 区为主
└── 否 → 无法调剂
    ├── 愿意二战 → 规划二战策略
    └── 不愿意 → 考虑就业/出国/其他出路
```

## 三、调剂流程与时间线

```
2月下旬        初试成绩公布
              ├→ 立刻判断是否需要调剂
              └→ 开始搜集调剂信息

3月中旬        国家线公布
              ├→ 确认是否过线
              └→ 锁定调剂目标院校清单

3月中下旬      研招网调剂意向采集系统开放
              ├→ 填报意向（不等于正式志愿）
              └→ 院校可以在系统里看到你的意向

3月底-4月     调剂系统正式开放 ⚠️ 最关键阶段
              ├→ 前48小时是黄金窗口！！！
              ├→ 每次可填 3 个平行志愿
              ├→ 每个志愿提交后锁定 36 小时
              └→ 收到复试通知 → 24小时内确认

4月-5月       各校陆续组织调剂复试
              ├→ 可以参加多所院校复试
              ├→ 复试通过 → 收到待录取通知
              └→ 只能接受 1 个录取

5月底         调剂基本结束
              └→ 少量院校有第二轮调剂（捡漏机会）
```

## 四、调剂信息搜集

### 优先渠道

| 渠道 | 优先级 | 说明 |
|------|--------|------|
| **研招网调剂系统** | 🔴 最高 | 官方唯一通道，所有调剂最终都要通过这里 |
| **目标院校研究生院官网** | 🔴 最高 | 部分院校在系统开放前就发布调剂公告 |
| 院校招生办电话 | 🟡 重要 | 直接问"今年 XX 专业预计有名额吗？对本科有要求吗？" |
| 学长学姐人脉 | 🟡 重要 | 能获取内部信息（导师态度、往年调剂生源质量） |
| 小木虫论坛 | 🟢 参考 | 有导师直接发帖招调剂生 |
| 考研帮 APP | 🟢 参考 | 汇总调剂信息，但可能有延迟 |
| 考研 QQ 群/微信群 | 🟢 参考 | 信息碎片化严重，仅作辅助 |

### 调剂信息核心关注点

对每个候选调剂院校，确认：
1. 今年预计有多少调剂名额？（比往年是增加还是减少）
2. 对本科层次有要求吗？（写"优先"= 985/211 优先，写"应"= 硬性要求）
3. 调剂复试时间？（避免与第一志愿复试冲突）
4. 调剂复试考什么？（通常和第一志愿复试不同，备考时间可能只有 3-7 天）
5. 对初试成绩有最低要求吗？（部分热门调剂院校有内部划线）

## 五、调剂策略

### 院校分类

#### 🚀 冲刺调剂档（初试分较高，过 A 区线 30+）

| 院校类型 | 特点 | 策略 |
|---------|------|------|
| 985 冷门专业/新设专业 | 竞争相对小 | 关注新设硕士点，第一年通常缺额 |
| A 区 211 部分专业 | 部分方向缺额 | 提前联系导师，表达诚意 |
| 中国科学院各研究所 | 部分所缺额，学术资源好 | 关注各地分所，不一定在北京上海 |

#### 🎯 稳妥调剂档（过 A 区线 10-30 分）

| 院校类型 | 特点 | 策略 |
|---------|------|------|
| **同城降档** | 同一城市的次一档院校 | 优先推荐——生活成本不增加，地域熟悉 |
| A 区双非强势专业 | 专业实力强但学校牌子一般 | 就业角度看，强专业 > 弱学校 |
| B 区 211 | 云南、贵州、广西、甘肃、新疆等 | 211 牌子在考公/落户中仍有价值 |

#### 🛡️ 保底调剂档（擦线过国家线）

| 院校类型 | 特点 | 策略 |
|---------|------|------|
| B 区双非 | 竞争最小 | 确保有学上 |
| 新增硕士点 | 第一年招生，知名度低 | 信息不对称的机会 |
| 非全日制 | 部分院校允许全日制调非全 | 如果只为了学历提升 |

## 六、调剂志愿填报技巧

### 3 个平行志愿怎么分配？

| 策略 | 志愿 1 | 志愿 2 | 志愿 3 |
|------|--------|--------|--------|
| 激进 | 冲刺 | 冲刺 | 稳妥 |
| 均衡 ✅ | 冲刺 | 稳妥 | 保底 |
| 保守 | 稳妥 | 稳妥 | 保底 |

### 关键规则

- 每个志愿提交后有 **36 小时锁定期**——期间不能修改
- 36 小时后未被查看或未收到通知 → 可以修改换学校
- 收到复试通知 → **24 小时内确认**，逾期作废
- 可以同时接受多个复试通知
- 但最终只能接受 **1 个待录取通知**
- 收到待录取通知后，通常有 2-6 小时的确认时间——**超时作废！**

### ⚠️ 黄金 48 小时

调剂系统开放后的前 48 小时是**每年最关键的窗口**：
- 大量院校会在这个时间段发布调剂信息
- 好学校的名额在前 48 小时基本被抢完
- 建议：凌晨系统一开放就立刻填（部分学校凌晨就发布）
- 提前准备好 3-5 个目标院校清单，系统一开按清单填

## 七、调剂复试准备

调剂复试的特点是：**通知到考试之间的时间极短（通常 3-7 天）**。

### 提前准备（在等调剂系统开放前就做好）

- [ ] 准备多份针对不同院校的简历（根据院校方向调整侧重点）
- [ ] 英文自我介绍练熟（1 分钟版 + 2 分钟版各一个）
- [ ] 整理调剂院校清单（Excel 表：院校名称/专业/预计名额/联系方式/截止日期/状态）
- [ ] 准备常见面试问题的回答（见 re-examination.md）
- [ ] 准备好政审表模板（不同院校格式可能不同）

### 收到调剂复试通知后的 72 小时

```
第 1 天：确认参加 → 订交通住宿 → 查找该院校复试经验帖 → 下载复试指定教材
第 2 天：突击专业课重点 + 模拟面试 2 次
第 3 天：出发/到达 → 现场踩点 → 早睡
```

## 八、常见调剂心理误区

| ❌ 误区 | ✅ 纠正 |
|--------|--------|
| "调剂 = 失败" | 每年大量学生通过调剂上岸，包括很多最终就业很好的人。调剂只是路径不同 |
| "只调 985" | 调剂 985 比一志愿考 985 更难——名额极少、本科要求极高 |
| "等第二轮捡漏" | 第二轮调剂名额极少且大多是冷门专业/偏远院校。**优先第一轮** |
| "同时接受 2 个录取" | 系统不允许——接受 1 个后，其他自动作废 |
| "先接受差的再等好的" | 一旦确认待录取 = 调剂结束，不能再参加其他学校复试 |
| "不调非全" | 如果你是为了学历+工作经验，非全日制是合理选择。关键看你的职业规划 |

## ⚠️ 关键原则
- 调剂拼的是**信息搜集速度 + 决策果断度**——犹豫就会败北
- 如果用户拒绝调剂 → 尊重选择，转二战规划或就业指导
- 调剂系统操作有时间限制——必须反复强调确认时间节点
- 调剂不是降级——B 区 211 的学术资源和培养质量可能不输 A 区 985 的一般专业
- 所有调剂信息以研招网和院校官网为准——论坛/群聊的信息需要交叉验证
