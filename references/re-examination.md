# 复试指导

## 适用场景
用户已通过初试（或预估能通过），需要准备复试。**复试不是走过场——部分院校复试占比 50%，初试高 30 分也可能被翻盘。**

---

## 一、复试流程概述

```
资格审查 → 专业课笔试（如有）→ 综合面试（核心）→ 等待结果
```

### 各环节权重（因校而异，以目标院校复试办法为准）

| 环节 | 常见占比 | 关键点 |
|------|---------|--------|
| 专业课笔试 | 30%-50% | 部分院校只面试不笔试，务必确认 |
| 综合面试 | 50%-70% | 含自我介绍 + 专业问答 + 英文口语 + 自由提问 |
| 英语听力/口语 | 5%-10% | 单独测试或融入面试中 |

---

## 二、复试准备清单

### 材料准备（出分后 3 天内完成）

| 材料 | 份数 | 备注 |
|------|------|------|
| 身份证原件 + 复印件 | 1+2 | — |
| 准考证 | 1 | 初试那张不要扔 |
| 本科成绩单 | 2 | 教务处盖章（往届生可去档案所在地复印） |
| 四六级成绩单 | 复印件 | — |
| 获奖证书/论文/专利 | 复印件各 2 份 | 只带含金量高的，学生会/社团类的基本没用 |
| 个人简历 | 5 份（彩色打印） | 1 页 A4，重点突出科研和竞赛 |
| 政审表 | 1 | 应届找辅导员，往届找档案所在地/街道办 |
| 学籍在线验证报告 | 1 | 学信网下载 |

### 面试材料准备

**个人简历（核心材料）**：
- 模板：简洁干净，不要花哨设计
- 内容优先级：科研经历 > 竞赛获奖 > 项目经历 > 实习 > 课程成绩 > 社团
- ❌ 不要写：精通 XXX（面试官追问 3 个问题就可能露馅）
- ✅ 写：熟悉 XXX，掌握 XXX，了解 XXX（给自己留余地）

**自我介绍**：
- 中文版 1-2 分钟（约 200-300 字）
- 英文版 1-2 分钟
- 核心公式：我是谁（30 秒）+ 我做了什么（60 秒）+ 为什么来这里（30 秒）

---

## 三、联系导师

### 时机
- **最佳时机**：初试出分后的 **24 小时内**（导师邮件会爆满，先发先被看到）
- 如果分数不确定能否进复试 → 等复试线公布再发
- **不要**初试前联系（没分数没意义）

### 邮件模板

```
主题：202X 年考研复试申请——XXX（你的名字）——初试 XXX 分

尊敬的 XX 老师：

您好！我是 XX 大学 XX 专业的 XXX。202X 年考研初试成绩为 XX 分（政治 XX，英语 XX，数学 XX，专业课 XX）。

我对您在 [具体研究方向，如"分布式存储系统"] 领域的研究非常感兴趣，特别是您近期发表的《[论文标题]》（[期刊/会议名]，202X）中关于 [具体技术点] 的研究给了我很大启发。

本科期间，我 [具体亮点：参与了 XX 项目/获得了 XX 竞赛奖项/完成了 XX 毕业设计]，具备一定的 [相关技能] 基础。

附件是我的个人简历。如果有机会进入您的课题组学习，我将全力以赴。

期待您的回复。祝您工作顺利！

学生：XXX
电话：XXXXXXXXX
日期：202X年X月X日
```

### ⚠️ 联系导师注意事项
- **同一学院一次只联系一位导师**——导师之间会交流，群发邮件的印象很差
- 如果 3 个工作日未回复 → 可以联系下一位
- 被婉拒后换下一位 → 不用纠结，导师名额有限是正常的
- **不要打电话或加微信**——邮件是唯一得体的初次联系方式
- 不要同时给导师发短信/打电话催回复

## 四、面试准备

### A. 中文自我介绍（1-2 分钟）

**结构模板**：
```
[30秒] 基本信息：各位老师好，我叫 XXX，来自 XX 大学 XX 专业。
[60秒] 核心亮点：本科期间，我主要做了两件事——
        1. [科研/竞赛/项目经历]（数字化："参与国家级大创项目，完成 XX 模块"）
        2. [与报考方向相关的经历]（关联化："这也促使我对 XX 方向产生了浓厚兴趣"）
[30秒] 读研动机：因此我希望在贵校 XX 方向深造，特别是对 XX 老师的研究方向非常关注。
        谢谢各位老师！
```

### B. 英文自我介绍（1-2 分钟）

**关键**：流利 > 高级词汇。用简单的句子，确保不会卡壳。

```
Good morning/afternoon, professors.

My name is [Name], and I'm from [University], majoring in [Major].

During my undergraduate study, I [achievement 1: participated in ... project,
where I learned ...]. I also [achievement 2: won ... competition / published ...].

I'm particularly interested in [research direction], which is why I applied for
[Target University]. I've read some papers by Professor [Name], and I'm very
impressed by the work on [specific topic].

That's all. Thank you for your time.
```

> 英文自我介绍控制在 1-1.5 分钟最好，不要超过 2 分钟。

### C. 常见面试问题 + 回答框架

| 问题 | 考察点 | 回答框架 |
|------|--------|---------|
| 为什么选择我们学校/专业？ | 报考动机 + 了解程度 | 对 XX 研究方向的兴趣 + 了解到贵校在 XX 领域的优势 + 特别关注了 XX 老师的工作 |
| 介绍你的本科毕业设计/论文 | 研究能力 | 背景（1句）+ 做了什么（2句）+ 你的贡献（1句）+ 收获（1句） |
| 本科期间印象最深的课程/项目？ | 学习深度 | 选一个与报考方向相关的项目，说清楚：问题→方案→结果 |
| 读过哪些专业书籍/论文？ | 学术视野 | 说 1-2 本/篇，能说清楚核心观点，不要只说书名 |
| 你的优缺点？ | 自我认知 | 优点说与科研相关的（细心、有耐心、善于总结）；缺点说真实但不致命的（如"公开演讲经验少，但正在练习"） |
| 如果没录取怎么办？ | 抗压能力 | "会先反思不足，然后考虑调剂或二战。但我对贵校的向往是非常真诚的" |
| 对哪个研究方向感兴趣？为什么？ | 学术规划 | 说出具体方向 + 1 个你了解的技术细节 + 为什么感兴趣 |
| 有读博打算吗？ | 学术规划 | 诚实回答。如果没想好，说"目前倾向于读博但不排除就业，会在研一深入接触研究后再做决定" |

### D. 专业课抽题应对

- 抽到完全不会的题 → **诚实承认 + 展示相关了解**："这个题目我确实不太熟悉，但据我所知，相关的内容是 XX。复试后我会补上这一块。"
- 抽到半会的题 → 先说确定的部分，不确定的部分标注"这是我目前的理解，可能不够全面"
- ❌ 千万不要不懂装懂——面试官几乎一定会追问，一旦暴露 = 诚信问题
- ✅ 展示思考过程——即使答案不完美，好的思考路径也能加分

## 五、面试礼仪与细节

### 着装
- 理工科：干净整洁为主，不需要正装（穿正装反而奇怪）
- 经管/文科：偏商务休闲
- **核心**：看起来像"认真的学生"，而不是"来面试 CEO"

### 面试当天
- 提前 30 分钟到考场
- 进门敲门 → 问好 → 等老师示意入座 → 坐 1/3 椅子 → 保持自然眼神交流
- 回答问题前先思考 3 秒（停顿 = 思考，不是紧张）
- 每道题回答控制在 1-2 分钟，不要超 3 分钟
- 结束时致谢 → 轻轻关门

### ⚠️ 面试避坑清单

| ❌ 绝对不要 | 原因 |
|-----------|------|
| 贬低本科学校/老师 | 面试官会认为你来了也会贬低这里 |
| 不懂装懂 | 诚信问题是直接淘汰的理由 |
| 背诵感太强的自我介绍 | 像是在背课文 = 准备不自然 |
| 与面试官争论 | 表达不同观点可以，争论 = 不尊重 |
| 迟到 | 几乎等于直接淘汰 |
| 简历造假 | 一旦被问穿 = 诚信污点 |
| 过度吹捧某位导师而贬低其他老师 | 面试是集体打分 |

## 六、复试前 3 天检查清单

- [ ] 确认复试时间+地点（官网多看几遍，最好截图保存）
- [ ] 订好交通+住宿（复试旺季学校周边酒店很快满）
- [ ] 所有材料原件+复印件装好文件袋
- [ ] 简历彩打 5 份（面试时递给每位老师）
- [ ] 自我介绍 中/英文各练 5 遍以上（对着镜子或录视频）
- [ ] 浏览目标导师近 3 年论文摘要（了解研究方向）
- [ ] 准备 3 个「想问导师的问题」（面试最后几乎都会被问"你有什么想问的"）
- [ ] 前一晚 23:00 前睡觉，不熬夜突击

---

## 输出格式

```
## 复试准备方案

### 时间线
| 时间 | 任务 |
|------|------|
| 出分当天 | 判断是否能进复试 → 联系导师（发邮件） |
| 出分后第 1-3 天 | 准备复试材料（成绩单/简历/政审等） |
| 出分后第 1-2 周 | 专业课复习（复试指定教材+导师论文） |
| 出分后第 2-3 周 | 模拟面试练习（找朋友/同学模拟） |
| 复试前 3 天 | 确认材料+订交通住宿+最终模拟 |

### 联系导师邮件模板
（根据用户情况填充模板，标注姓名/学校/分数/研究方向）

### 面试常见问题
（根据用户专业和方向，从题库中选择 5-8 个最可能被问到的问题 + 个性化回答框架）

### ⚠️ 个性化避坑提醒
（根据用户画像：跨专业 / 往届 / 在职 / 本科院校层次，给出针对性提醒）
```

## ⚠️ 关键原则
- 复试的关键不是"表现完美"，是"看起来值得培养"——展示学习能力和诚实比展示"什么都会"更重要
- 联系导师邮件不要群发，不要催，不要同时联系多人
- 简历 1 页就够，超过 1 页的删——面试官不会翻页
- 如果用户初试分数边缘（刚过复试线），需要同步准备调剂（详见 adjustment-guide.md）
- 面试后 3 天可以发一封简短的感谢邮件给意向导师（一句话表示参加了面试+感谢机会即可）
