# 报名与确认指导

## 适用场景
用户进入 9-11 月，需要了解预报名、正式报名、网上确认全流程。**每年都有人因为报名失误而白备考一年——这些坑提前知道就能避免。**

---

## 一、报名全流程时间线

```
9月中旬        研招网发布《全国硕士研究生招生工作管理规定》
              ├→ 确认今年考试时间（通常12月倒数第二个周末）
              └→ 关注目标院校招生简章（9月陆续发布）

9月下旬        预报名（应届生）
              ├→ 约9月24日-27日，每天9:00-22:00
              ├→ 预报名数据有效，正式报名时无需重复
              └→ 目的：分流压力 + 让应届生提前占考点

10月中上旬     正式报名
              ├→ 约10月5日-25日，每天9:00-22:00
              ├→ 所有考生（应届+往届）都必须在此期间完成报名
              ├→ 预报名已完成的考生可以修改信息
              └→ ⚠️ 这是唯一报名窗口——错过=等一年

11月初         网上确认（原现场确认）
              ├→ 约11月1日-5日（各省时间略有差异）
              ├→ 上传照片+证件+确认信息
              └→ ⚠️ 不确认=报名无效！

12月中旬       打印准考证
              ├→ 考前约10天开放下载
              └→ 多打几份备用（3-5份），手机里存电子版
```

---

## 二、预报名 vs 正式报名

| | 预报名 | 正式报名 |
|------|--------|---------|
| 时间 | 9月24-27日 | 10月5-25日 |
| 面向 | 主要是应届生（部分省往届生也可） | 所有考生 |
| 效力 | 与正式报名同等有效 | — |
| 考点选择 | **先到先得**——这是预报名的最大价值 | 热门考点可能已满 |
| 能否修改 | 正式报名期间可以修改 | 截止前可以修改 |

### ⚠️ 为什么一定要参加预报名？

**考点容量有限。** 热门城市（北上广深/武汉/南京等）的考点通常在预报名阶段就被抢光。如果等到正式报名，可能只剩郊区考点甚至异地考点——考试当天通勤 2 小时+会严重影响状态。

> 应届生：**预报名第一天就报**，不要等。考点 = 手速。

## 三、报名前需要准备的材料

### 应届生

| 材料 | 说明 |
|------|------|
| 身份证 | 确认有效期——如果考试前过期，现在就换 |
| 学生证 | 每学期注册章要盖全 |
| 学信网账号 | 提前注册并完成学籍验证（学信网→学信档案→学籍验证） |
| 近期免冠照片（电子版） | 白底或蓝底，按要求尺寸——网上确认时上传 |
| 个人信息确认 | 姓名/身份证号/民族/手机号——一个字都不能错 |

### 往届生（额外需要）

| 材料 | 说明 |
|------|------|
| 毕业证 + 学位证 | 原件 + 复印件 + 电子扫描件 |
| 档案所在地 | 确认档案在哪个单位（人才市场/原学校/工作单位） |
| 工作证明/社保证明 | 如果选择在工作所在地考试（而非户籍地）——部分省份要求 |

## 四、报名信息填写 —— 最容易出错的地方

### 🔴 致命级错误（导致报名无效）

| 错误 | 后果 | 正确做法 |
|------|------|---------|
| 身份证号填错 | 无法确认，报名无效 | 再三核对！数字+字母全部确认 |
| 报考点选错 | 无法考试 | 应届生选**学校所在地**考点；往届生选**户籍地**或**工作地**（需满足当地要求） |
| 学历校验不通过 | 报名失败 | 提前在学信网做学籍/学历验证，有问题立即联系学校 |
| 忘记网上确认 | 报名无效 | 设闹钟！11月初！不确认 = 今年白给 |

### 🟡 严重级错误（可修改但麻烦）

| 错误 | 后果 | 正确做法 |
|------|------|---------|
| 报考院校/专业代码填错 | 考错学校/专业 | 报名截止前可改；截止后无法改 |
| 考试科目选错 | 复习的和考的不一样 | 核对目标院校招生简章上的科目代码 |
| 手机号填错或停用 | 收不到重要通知 | 保证手机号用到录取结束（次年6月） |
| 通讯地址填错 | 录取通知书收不到 | 填一个长期有效的地址（家里或学校） |

## 五、网上确认（11月初）

网上确认是**必须完成的步骤**——不确认 = 没报名。

### 网上确认流程

```
1. 登录研招网 → 网上确认系统
2. 阅读网报公告 → 确认诚信承诺书
3. 上传本人照片（按要求：免冠/白底或蓝底/不带眼镜/不化妆过度）
4. 上传身份证正反面照片
5. 上传手持身份证照片
6. 应届生：上传学生证；往届生：上传毕业证/学位证
7. 选择报考点要求的其他材料
8. 提交 → 等待审核（通常1-3个工作日）
9. 审核通过 → ✅ 报名完成
```

### ⚠️ 网上确认常见拒审原因

| 拒审原因 | 如何避免 |
|---------|---------|
| 照片不符合要求 | 不要用生活照/自拍/美颜过度的照片；去照相馆拍证件照 |
| 手持身份证照片不清晰 | 身份证上的信息必须清晰可读；光线要好 |
| 学生证缺少注册章 | 提前去院办补盖 |
| 学籍/学历验证不通过 | 提前在学信网验证，有问题的联系学校教务处 |

> 上传后每天登录查看审核状态——如果不通过，系统会告诉你原因，修改后重新上传。不要提交了就以为完事了。

## 六、准考证（12月中旬）

### 下载与打印

- 登录研招网 → 准考证下载 → 用 A4 白纸打印
- **多打几份**（3-5 份），分开放（书包/笔袋/住处各一份）
- 手机里存 PDF 电子版
- 正反面**不能有任何涂写**（包括草稿——会被视为作弊）

### 准考证上的关键信息

拿到准考证后立刻确认：
- [ ] 姓名/身份证号/照片
- [ ] 考试时间（每天几点到几点）
- [ ] **考点地址**（具体的学校和教室——不是"XX市"，是"XX学校XX楼XX教室"）
- [ ] 考点是否有特殊要求（如自带文具/不能带手表/提前多久到场）

### ⚠️ 确认考点后立刻做的事

**订酒店。** 考点一出，周边酒店会在 2 小时内被订光。标准：
- 步行 15 分钟以内（打车可能堵车，走路最稳妥）
- 不要订临街的房间（噪音影响休息）
- 如果可以，提前一天入住适应环境

## 七、报名期间的心理管理

9-10 月是考研的第一个「弃考高峰」——报名本身就会触发大量焦虑。

### 常见心理波动 + 应对

| 心理波动 | 应对 |
|---------|------|
| "报名了，感觉来不及了" | 报名 ≠ 考试。你还有 2-3 个月。报名只是确认你要考——不是判定结果 |
| "要不要换学校？" | 10 月 25 日前可以改。但必须以**模考数据**做决策，不以焦虑做决策（详见 mock-exam-adjustment.md） |
| "别人都报了好学校，我是不是报低了？" | 别人的人生不关你的事。你报的学校是理性分析后的选择——信任之前的自己 |
| "填信息填到怀疑人生" | 正常——每个人填报名表都紧张。核对三遍，截图保存，提交后就可以忘了 |

```
- 需要准备的材料：[清单]

### 关键时间提醒
| 时间 | 事项 | 你需要做的 |
|------|------|-----------|

### ⚠️ 你最容易踩的坑
（根据用户类型给出 3-5 条个性化提醒）

### 报名信息核对清单
- [ ] 身份证号
- [ ] 报考院校+专业代码
- [ ] 考试科目代码
- [ ] 报考点
- [ ] 手机号
- [ ] 通讯地址
```
