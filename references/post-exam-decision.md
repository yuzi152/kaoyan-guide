# 出分与决策

## 适用场景
初试已考完，成绩即将公布或已公布。用户处于「等分焦虑 → 查分恐慌 → 不知道下一步怎么办」的阶段。**出分后的 72 小时是决策黄金窗口——每一个小时都在消耗调剂名额和导师名额。**

---

## 一、出分前：准备工作（查分前 1 周）

### 提前做好的事

| 事项 | 说明 |
|------|------|
| **确认查分渠道** | 各省教育考试院官网 + 研招网。提前收藏网址，确认账号密码能登录 |
| **准备复试材料草稿** | 个人简历、中英文自我介绍、联系导师邮件模板——出分前就写好，出分后只需填分数即可发出 |
| **整理目标院校去年数据** | 去年复试线、去年录取最低分、去年复试时间、去年调剂公告发布时间 |
| **确认备选调剂院校清单** | 至少列出 5-8 所可能的调剂院校（如果第一志愿不稳） |
| **心理准备** | 给自己做个预期：最好的情况、最坏的情况、最可能的情况各是什么 |

---

## 二、查分当天

### 查分时需要记录的数据

光看总分不够——需要逐科记录：

```
总分：XXX
政治：XX  英语：XX  数学：XX  专业课：XX

去年目标院校复试线：XXX
去年目标院校录取最低分：XXX
去年国家线（A区）：XXX /（B区）：XXX

初步判断：
[ ] 总分比去年复试线高 15+  → 稳了，全力准备复试
[ ] 总分比去年复试线高 0-15  → 擦线，复试+调剂双线并行
[ ] 总分比去年复试线低     → 等国家线，主力调剂
[ ] 总分比去年 B 区线低    → 考虑二战或就业
```

### 查分后立刻做的事（按优先级）

| # | 事项 | 时限 | 为什么急 |
|---|------|------|---------|
| 1 | **联系导师** | 查分后 **2 小时内** | 导师邮箱当天会爆满——先发先被看到。邮件模板见 re-examination.md |
| 2 | **对比近 3 年数据** | 查分后 1 天内 | 只看去年不够——前年和大前年的波动范围决定你的判断准确度 |
| 3 | **决定策略模式** | 查分后 2 天内 | 单线（只准备复试）还是双线（复试+调剂并行）——见下方决策矩阵 |
| 4 | **复核分数（如需要）** | 各省规定时间内（通常 3-5 天） | 如果某科分数明显低于预期——但复核成功率很低，不要抱太大希望 |

## 三、决策矩阵：三种情况 × 三种策略

### 情况 A：稳进复试（总分 > 去年录取最低分）

```
策略：单线冲刺复试 ✅

行动清单：
├── 当天：发联系导师邮件（必须当天）
├── 第 1-3 天：准备复试材料（简历/成绩单/政审）
├── 第 1-2 周：全力准备复试专业课 + 面试
├── 复试前 1 周：模拟面试每天 1 次
└── 只关注目标院校复试通知，不分散精力收集调剂
```

### 情况 B：擦线（去年复试线 ± 10 分内）

```
策略：复试 + 调剂双线并行 ⚠️

这是最累但最必要的模式——复试差几分被刷的人，
如果有调剂备选 = 仍然上岸；如果没有 = 落榜。

双线时间分配：复试 70% + 调剂 30%

行动清单：
├── 当天：发联系导师邮件（目标院校 + 2-3 所调剂院校）
├── 第 1-7 天：主要精力在复试准备
├── 同时：每天花 1 小时刷新调剂信息 + 联系调剂院校招生办
├── 重点关注：同城降档院校（专业课相同 → 复试准备不浪费）
└── 如果目标院校复试在前：全力复试，复试结束后立刻转调剂
    如果调剂院校复试在前：先拿下调剂 offer，再冲目标院校
```

### 情况 C：未达复试线（但过国家线）

```
策略：全力调剂 🔴

行动清单：
├── 当天：放弃目标院校幻想，立刻转入调剂模式
├── 第 1-2 天：锁定调剂目标院校清单（至少 10 所）
├── 第 3-7 天：联系调剂院校导师 + 招生办
├── 每天刷新研招网 + 目标院校官网（早上/中午/晚上各一次）
├── 准备多份简历（根据不同院校方向定制侧重点）
└── 如果 A 区无望 → 立刻扩展到 B 区院校
```

> 📎 调剂的详细策略见 [调剂指导](adjustment-guide.md)

### 情况 D：未过国家线

```
策略：二战或就业

├── 先给自己 1 周的冷静期（不要查分当天就决定）
├── 做失败原因分析（详见 retake-decision.md）
├── 与家人/朋友深入讨论
└── 1 周后做出决定：二战/就业/其他出路
```

> 📎 二战决策详见 [二战决策指导](retake-decision.md)

## 四、多线程并行策略（擦线考生专用）

擦线考生面临最复杂的情况：一手准备复试，一手收集调剂，一手还要应对心理压力。

### 如何不分裂？

```
上午（09:00-12:00）= 复试时间 🔴
  → 准备目标院校复试专业课 + 模拟面试
  → 这是主战场——不能因为"可能进不了复试"就放弃

下午（14:00-16:00）= 复试时间 🔴
  → 继续复试准备

下午（16:00-17:00）= 调剂时间 🟡
  → 刷新调剂信息 + 联系 1-2 所调剂院校
  → 每天 1 小时——多了会分散复试精力
  → 少了会错失调剂窗口

晚上 = 灵活时间 🟢
  → 根据当天情况：复试弱就补复试，调剂有消息就追调剂
```

### 擦线考生最容易犯的错误

| ❌ 错误 | ✅ 正确 |
|--------|--------|
| "我可能进不了复试，全力调剂算了" | 目标院校复试 > 调剂。你的第一志愿是最优选择——不到最后一刻不要放弃 |
| "我一定要冲目标院校，调剂不考虑" | 擦线 = 概率不是 100%。没有备用方案 = 把自己逼入绝境 |
| 同时联系 5-6 个调剂导师 | 精力分散。精选 2-3 个最靠谱的调剂方向即可 |
| 每天花大量时间刷调剂群信息 | 焦虑的源头。每天固定 1 小时刷调剂信息就够了 |

## 五、分数复核

如果某科分数**明显低于你的预期**（比如英语平时模考 70+，出分 45），可以申请复核。

### 复核流程

```
1. 查分后 3-5 天内 → 登录省教育考试院官网
2. 找到"成绩复核"入口 → 填写申请
3. 复核范围：仅核查有无漏评、加分错、登分错
   → 不重新评卷！不查评分宽严！
4. 复核结果：通常 1-2 周内通知
```

### ⚠️ 复核的真实情况

- 复核成功率**极低**（< 1%）——阅卷流程本身有多重校验
- 复核**不会降分**——不用担心越查越低
- 如果分数与预期差距在 5 分以内 → 大概率是你高估了自己，不是改错了
- 如果单科差距在 20 分以上 → 可能是答题卡涂错位——这种情况复核有可能发现

> 复核期间不要停止准备复试或调剂——复核只是买个心安，不能作为拖延决策的理由。

## 六、心态管理（最焦虑的 2 周）

出分前后是整个考研过程中**心态最脆弱的时期**。从初试结束到出分之间是「漫长等待」，出分后是「激烈抉择」。

### 常见心理波动

| 阶段 | 感受 | 应对 |
|------|------|------|
| 等分期 | 焦虑、反复回想考试细节、失眠 | 做具体的事——准备复试材料、写简历、读导师论文。有事做就不瞎想 |
| 出分瞬间 | 手抖、不敢看、看完不敢相信 | 先看总分再看单科——给自己 10 分钟消化，不要立刻做任何决定 |
| 出分后 24h | 反复对比排名和分数线、狂刷群聊 | 对比 3 个可靠数据源就停。刷群只会看到「高分晒分」增加焦虑 |
| 出分后 3-7 天 | 等待复试线/国家线的煎熬 | 这段时间按「擦线策略」做准备——不管结果如何，先行动起来 |
| 看到别人高分时 | "我是不是很失败" | 只有高分的人才会出来晒——沉默的大多数和你差不多。比较没有意义 |

### 出分后的社交建议

- ✅ 告诉 1-2 个最亲近的人（家人/伴侣/最好的朋友）
- ❌ 不要发朋友圈/在群里晒分（不管高分低分——高了招人烦，低了自己难受）
- ❌ 出分后 48 小时内不刷考研群（群里全是焦虑放大器）
- ✅ 如果身边有同样考研的朋友——互相支持但不比较分数
