# 二战决策指导

## 适用场景
初试未过国家线、复试被刷且调剂无果、或调剂结果不理想。用户面临「要不要再考一年」的重大决定。**二战不是简单地把去年做的事再做一遍——是需要重新分析、重新定位、重新出发的严肃决策。**

---

## 一、决策框架：四个问题

在决定二战之前，花 3 天时间认真回答以下四个问题。不要出分当天就决定。

### 问题 1：去年为什么失败？

| 失败原因类型 | 具体表现 | 二战可改善吗？ |
|------------|---------|-------------|
| **时间不足** | 启动太晚、在职太忙、被实习/毕设挤占 | ✅ 可以——二战有完整的一年 |
| **方法不对** | 只看视频不刷题、资料不对、跟错老师 | ✅ 可以——换个方法就是质的提升 |
| **目标过高** | 双非冲 C9、跨专业冲 985、报录比极高的专业 | ✅ 可以——降低目标或换个方向 |
| **单科拖腿** | 数学/英语没过单科线 | ✅ 可以——集中突破薄弱科 |
| **心态崩溃** | 考前/考场过度焦虑、弃考、发挥失常 | ⚠️ 不确定——取决于能否找到焦虑源并解决 |
| **不可抗力** | 生病、家庭变故、考场突发状况 | ⚠️ 不确定——今年未必不会再发生 |
| **已经尽力但差距太大** | 全力备考但总分与目标差 50+ | 🔴 很难——说明目标与能力差距过大，需大幅调整 |

### 问题 2：如果再来一年，能改变什么？

不要笼统地说「我会更努力」——需要具体到：
- 去年数学只刷了 1000 题的 60% → 今年能完成多少？
- 去年政治 11 月才开始背 → 今年什么时候开始？
- 去年没有参加模拟考 → 今年能做到每月一次模考吗？
- 去年每天实际学习 6 小时 → 今年能稳定做到 8 小时吗？

### 问题 3：二战的机会成本是什么？

| 成本 | 估算 | 你能否承受？ |
|------|------|------------|
| **一年时间** | 22-25 岁中宝贵的一年 | |
| **经济成本** | 生活费 + 资料费 + 可能的租房费 ≈ 3-8 万 | |
| **心理压力** | 二战的心理压力显著大于一战 | |
| **职业起步延迟** | 同龄人可能已工作一年、读研一年 | |
| **再次失败的风险** | 如果二战又没考上？ | |

### 问题 4：有没有不考虑二战的替代方案？

| 替代方案 | 适合谁 |
|---------|--------|
| **先工作，边工作边考** | 需要经济独立、不能脱产的人 |
| **换个更容易的专业/学校再考** | 坚持要读研但目标过高的人 |
| **申请境外/海外研究生** | 英语好、家庭经济条件支持 |
| **直接就业** | 对读研本身不坚定、只是为了逃避就业的人 |
| **考公/考编** | 本身就考虑过体制内的人 |

## 二、是否适合二战？快速自检

以下 5 条中，**绿色 ≥ 3 条 → 适合二战**；红色 ≥ 3 条 → 慎重。

| # | 陈述 | 你的答案 |
|---|------|---------|
| 1 | 去年有**明确可改善**的问题（不是"运气不好"） | ✅ / ❌ |
| 2 | 考研是为了**自己的目标**（不是父母期望/逃避就业） | ✅ / ❌ |
| 3 | 家庭**支持**二战（经济+情感） | ✅ / ❌ |
| 4 | 有**稳定的学习环境**（家里/租房/自习室） | ✅ / ❌ |
| 5 | 即使二战失败，有**最坏的备选方案**且能接受 | ✅ / ❌ |

## 三、如果决定二战 —— 四个调整

### 调整 1：重新分析失败原因（要具体，不要笼统）

```
❌ 笼统分析（没用）：
"我去年不够努力"、"数学太难了"、"运气不好"

✅ 具体分析（有用）：
"我去年 7 月才开始做数学真题，只做了近 5 年的 3 套。
正确率 55%，主要集中在极限计算和线代方程组这两章出错。
今年需要：① 5 月开始跟李永乐线代讲义系统学 ② 习题集
正确率目标 70% ③ 近 10 年真题全部限时模考"
```

### 调整 2：重新评估目标（不要原样再冲一次）

| 去年情况 | 二战建议 |
|---------|---------|
| 总分与目标差 < 15 分 | 可以保持原目标，微调方法 |
| 总分与目标差 15-30 分 | 建议降低半档目标（985→211 或学硕→专硕） |
| 总分与目标差 30-50 分 | 建议降一档目标 + 换专业课更匹配的院校 |
| 总分与目标差 > 50 分 | 强烈建议重新择校——不是你不努力，是目标与基础差距过大 |
| 专业课单科不过线 | 优先选专业课科目相同的院校（学的不用推翻重来） |
| 数学单科不过线 | 考虑换不考数学的专业，或换数二/数三 |

### 调整 3：重新设计学习方案

| 去年踩的坑 | 今年怎么改 |
|-----------|-----------|
| 启动太晚 | 现在就开始——不要再等到"过完年再说" |
| 刷题太少 | 选定一本习题集，设定完成率目标（如 880 题完成 90%） |
| 只学不考（不模考） | 从第 3 个月开始，每月至少 1 次完整模考 |
| 孤军奋战 | 找一个靠谱研友互相打卡（不是互相焦虑的那种） |
| 资料太多太杂 | 选定一套主资料跟到底——去年你已经知道哪些资料是浪费钱了 |
| 没有阶段性目标 | 设置每月里程碑 + 每周小目标（见 progress-tracking.md） |

### 调整 4：重新安排时间线

```
现在（出分后 1 个月）     冷静期 + 决策期
├── 做失败原因分析
├── 与家人沟通
├── 如果决定二战：重新择校 + 制定新计划
└── 如果决定就业：开始投简历/准备春招

3月-6月                   基础阶段
├── 二战的你不需要从头学一遍——重点补去年的薄弱章
├── 数学：直接刷薄弱章的题（不要又从头看视频）
├── 英语：词汇保持每天复习（去年背过的不会全忘）
└── 408/专业课：去年学过一遍，今年重点是真题深度

7月-9月                   强化阶段
├── 数学：主攻真题 + 一本习题集
├── 408：真题按年份限时模考
├── 英语：阅读 + 作文模板开始
└── 政治：7 月开始（比去年早一点）

10月-12月                  冲刺阶段
├── 第二阶段刷真题 + 模考
├── 政治：肖八 → 肖四
└── 心理调节（二战心理压力比一战大——提前注意）
```

## 四、二战的心理建设

### 二战考生的独特心理挑战

| 挑战 | 应对 |
|------|------|
| "同龄人都在读研/工作了，我还在考试" | 人生不是同步的。一年在职业生涯中只是一瞬间。用这一年的投资换三年的研究生——值不值，你自己判断 |
| "万一又考不上怎么办" | 设好最坏预案（就业/考公/换方向）——最坏的情况你能接受，就不会被恐惧绑架 |
| "去年学了那么久还要重来" | 去年的知识不是白费的——二战的知识基础比一战好得多。你不是从零开始，是站在去年的肩膀上前进 |
| 家人的态度（支持 or 质疑） | 坦诚沟通：告诉他们你的分析、你的调整、你的备选方案。有具体计划比"我就是想再考一次"更有说服力 |
| 孤独感 | 找一个二战的研友组队（线上也可）——你们能互相理解，一战的人理解不了二战的感受 |
| 怀疑自己的决定 | 每个月检查一次进度——如果进度符合计划 = 你的决定是对的；如果严重落后 = 检查是否需要调整而非全盘否定 |

### 二战期间的心理规则

1. **不比较**：不跟一战的人比进度（你已经学过一遍），不跟工作的人比收入（你选的是另一条路）
2. **不过度社交**：少发朋友圈、少跟太多人解释"为什么还在考研"——解释成本太高
3. **每月一次「初心回顾」**：写下你为什么要考研——在焦虑的时候回看，提醒自己为什么出发
4. **给失败留一条后路**：坦然接受"这次也可能失败"——不是消极，是让自己不被恐惧压垮

## 五、如果决定不二战

不二战 ≠ 失败。以下是不二战之后的路径：

| 路径 | 行动计划 |
|------|---------|
| **直接就业** | 3-4 月春招是最后窗口——立刻开始投简历。计算机专业即使没考上研，有技术也能找到不错的工作 |
| **先工作再考虑在职考研** | 找一份不加班的工作（国企/事业单位/外企某些岗位），利用晚上和周末备考 |
| **考公/考编** | 关注省考/事业单位招聘时间——有些岗位对专业有要求，计算机可选岗位较多 |
| **申请境外研究生** | 英语好+有预算 → 可以考虑澳洲/英国/香港的计算机硕士（申请周期和国内考研不同） |
