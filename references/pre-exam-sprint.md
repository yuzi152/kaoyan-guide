# 考前冲刺与临场策略

## 适用场景
距考试 < 1 个月，或准考证已打印，用户需要考前最后阶段的冲刺安排和考试两天的临场策略。**最后两周不是用来学新东西的——是用来把已有的东西稳稳拿住的。**

---

## 一、考前 14 天倒计时

### 倒计时第 14-8 天：收尾期

```
数学：
├── 最后一遍真题模考（选最近的 3 套，不全刷）
├── 错题本快速翻一遍（只看反复错的题）
├── 公式默写（每天 10 分钟——极限、积分表、微分方程解法、线代公式）
└── ❌ 不做新题、不做模拟卷

408：
├── 四科核心概念快速过一遍（每科 1-2 天）
├── 只看错题本——不做新题
├── 算法模板默写（DS 常用操作：链表反转/二叉树遍历/图的遍历/排序算法）
└── ❌ 不做新题

政治：
├── 肖四大题第 3 轮背诵
├── 选择题错题最后一遍
├── 时政选择题集中过一遍（约 100 道时政题）
└── ❌ 不做其他老师的押题卷（贪多嚼不烂）

英语：
├── 作文模板最终确认 + 每天默写
├── 阅读保持手感（每天 1 篇，不追求数量）
├── 核心词汇快速扫一遍
└── ❌ 不学新单词、不做模拟题
```

---

### 倒计时第 7-3 天：保温期

```
所有科目：
├── 不！做！新！题！——这条字最大
├── 只看：错题本 + 公式/概念 + 作文模板 + 肖四框架
├── 每天学习 6 小时左右（刻意减少，让大脑休息）
└── 调整作息——按考试当天的时间表起床和睡觉

具体动作：
- 每天上午 8:30-11:30 → 模拟考政治/数学的生物钟（只是坐在书桌前看错题，不是模考）
- 每天下午 14:00-17:00 → 模拟考英语/408 的生物钟
- 让大脑在这个时段处于最活跃状态
```

### 倒计时第 2-1 天：考前日

```
考前 2 天：
├── 上午：政治肖四框架默写（最后一遍）
├── 下午：英语作文模板默写（最后一遍）
├── 晚上：准备考试用品（见下文清单）

考前 1 天：
├── 去考点踩点（确认教室位置、厕所位置、从住处走到考点的时间）
├── 只看：错题本 + 公式 + 作文模板 + 肖四框架
├── 下午 5 点后：完全不看任何学习资料
├── 晚上：放松 → 检查物品 → 22:00 睡觉
```

## 二、考试用品清单

### 必带（缺一不可）

| 物品 | 数量 | 备注 |
|------|------|------|
| 身份证 | 1 | 有效期内的原件 |
| 准考证 | 3-5 份 | A4 打印，分开放，手机存 PDF |
| 0.5 黑色签字笔 | 3-5 支 | 同型号，提前试写确认出水流畅 |
| 2B 铅笔 | 2-3 支 | 涂答题卡专用（自动 2B 铅笔更方便） |
| 橡皮 | 1 块 | 干净的，不卷边 |
| 口罩 | 2-3 个 | 考点可能有要求 |

### 建议带

| 物品 | 备注 |
|------|------|
| 透明笔袋 | 把所有文具装在一起——考点通常要求透明 |
| 机械手表 | 指针式（非电子表）——部分考场没有钟！先确认考点是否允许带 |
| 水杯 | 透明、无标签、放在地上（不放在桌上——打翻毁答题卡 = 灾难） |
| 巧克力/能量棒 | 3 小时考试中途饿了会严重影响思维 |
| 纸巾 | 小包装 |
| 备用眼镜 | 戴眼镜的考生——万一坏了还有备用 |

### ⚠️ 绝对不要带

| 物品 | 后果 |
|------|------|
| 手机（关机也不能放身上） | 必须放在考场指定位置或直接不带 |
| 智能手表/手环 | 视为作弊工具 |
| 任何纸张/笔记本 | 考场提供草稿纸 |
| 涂改液/修正带 | 不允许使用——写错划掉重写即可 |
| 带字的文具/水杯 | 可能被视为作弊 |

## 三、各科答题策略

### 政治（第一天上午 8:30-11:30）

**时间分配**：

| 题型 | 题量 | 建议用时 | 策略 |
|------|------|---------|------|
| 单选 | 16 题 | 15-20 分钟 | 快速做，不纠结。不会的先标记，全做完再回来 |
| 多选 | 17 题 | 35-40 分钟 | 多选是拉分关键——不确定的选项宁愿不选（少选 = 0 分，多选 = 0 分） |
| 分析题 | 5 题 | 每道 25 分钟 × 5 = 125 分钟 | 每题写满答题区域的 80% 即可 |
| 检查 | — | 5-10 分钟 | 重点检查选择题答题卡——涂错位 = 全卷废 |

**分析题策略**：
- 马原：先写原理名称 + 方法论 → 再用材料中的话展开 → 最后点题
- 毛中特/史纲/思修/时政：按肖四框架 + 结合材料展开
- **分点作答更清晰**：①②③
- 字迹尽量清晰——阅卷老师 30 秒扫一题，看不清 = 低分
- 政治是第一科，考完不要对答案！中午好好休息，下午还有英语

---

### 英语（第一天下午 14:00-17:00）

**时间分配**：

| 题型 | 题量 | 建议用时 | 策略 |
|------|------|---------|------|
| 完形填空 | 20 题 | 10-15 分钟 | 放最后做——0.5分/题，性价比最低 |
| 阅读理解 | 4 篇 20 题 | 60-70 分钟（每篇 15-18 分钟） | 先读题再读文，带着问题找答案 |
| 新题型 | 5 题 | 15-20 分钟 | 英二新题型较简单，拿满分是目标 |
| 翻译 | 1 篇 | 20-25 分钟 | 英二翻译是整段翻译（非英一的 5 个句子），流畅 > 精确 |
| 小作文 | 1 篇 | 15-20 分钟 | 格式正确（书信/通知/备忘录）就有基本分 |
| 大作文 | 1 篇 | 30-35 分钟 | 套模板 + 结合题目具体内容，字数够+结构对=及格分以上 |
| 检查 | — | 5 分钟 | 答题卡 + 作文拼写 |

**关键策略**：
- 答题顺序：阅读 → 新题型 → 翻译 → 作文 → 完形（或作文提前到翻译之前）
- 阅读如果卡在某篇文章太久（>20 分钟），果断跳过——做完全部再回来
- 作文模板套用时一定要结合题目——完全照搬模板 = 低分

---

### 数学（第二天上午 8:30-11:30）

**时间分配**：

| 题型 | 题量 | 建议用时 | 策略 |
|------|------|---------|------|
| 选择 | 10 题（数二） | 40-50 分钟 | 每题约 4-5 分钟。先做会做的，不会的跳过不纠结 |
| 填空 | 6 题（数二） | 25-30 分钟 | 每题约 4-5 分钟。结果化简到最简形式 |
| 解答题 | 6 题（数二） | 100-110 分钟 | 每题约 17-18 分钟 |

**解答题策略**：
- **前 3 道解答题**通常难度适中，全力拿满分——这 35-40 分是基础盘
- **后 3 道**可能较难——能写多少写多少，不要求完整解
- **即使不会也写步骤**：数学阅卷按步骤给分。设未知数 1 分，列方程 1 分，代入 1 分——空着 = 0 分
- 第一问不会？跳过做第二问——第二问可能独立给分

**选择填空策略**：
- 特殊值法/排除法/代入法——选择填空不需要完整解题过程，能用技巧就用
- 答案通常是简单形式（整数/简分数）——如果算出来一个奇怪的分数，检查一下
- 选择题不会的——排除明显错误的后，猜一个概率最高的

---

### 408 计算机专业基础（第二天下午 14:00-17:00）

**时间分配**：

| 题型 | 题量 | 建议用时 | 策略 |
|------|------|---------|------|
| 选择题 | 40 题 | 50-60 分钟 | 每题约 1-1.5 分钟。不会的标记跳过，不在一道题上卡 3 分钟 |
| 综合题 | 约 7 题 | 110-120 分钟 | 每题约 15-17 分钟。先 DS（你最强的）→ 计组 → OS → 计网 |

**408 答题顺序策略**：
1. 先做 DS 综合题（你的最强项——蓝桥杯省二的算法底子，优先拿稳定分）
2. 再做计组综合题（数据通路/流水线是固定套路，练熟了就不难）
3. 再做 OS 综合题（PV 操作模板化，把模板一套就完事）
4. 最后做计网综合题（分值最小，放最后不怕时间不够）

**各题型策略**：
- 选择题不确定的 → 排除法排除 2 个，剩余 2 个猜 1 个（50% 概率），不空题
- 算法题 → 伪代码 + 思路描述也有分，不要纠结写完美 C 代码
- 计算题 → 步骤分比结果重要！公式写对 2 分，代入正确 2 分，结果错了也就扣 1-2 分
- 不会的综合题 → 写出你知道的所有相关内容——可能蹭到 1-2 分

## 四、考试两天的生存指南

### 第一天（政治+英语）

```
06:30  起床
07:00  简单早餐（不要吃太饱——血糖升高会犯困）
07:30  出发去考点（预留堵车时间）
08:00  到达考点，去趟厕所
08:30  政治开考
11:30  政治结束
       ↓
       不！要！对！答！案！
       不看群、不看微博、不看任何"政治答案"
       对答案除了制造焦虑毫无用处
       ↓
12:00  午餐（清淡，少油少辣——肠胃是第一生产力）
12:30  午休 20 分钟
13:00  翻一眼英语作文模板（只是保持熟悉感，不是突击）
13:40  出发去考点
14:00  英语开考
17:00  英语结束
       ↓
       同样不要对答案
       ↓
18:00  晚餐
19:00  翻一下数学公式 + 错题本（1 小时就够）
20:30  检查数学考试用品
22:00  睡觉
```

### 第二天（数学+408）

```
06:30  起床
07:00  早餐（同样清淡）
07:30  出发
08:00  到达考点
08:30  数学开考
11:30  数学结束
       ↓
       不要对答案。不管你感觉多差——继续考
       （很多人觉得数学考砸了结果成绩意外好）
       ↓
12:00  午餐+午休
13:00  翻一眼 408 核心概念（保持熟悉感）
13:40  出发
14:00  408 开考
17:00  408 结束
       ↓
       🎉 你考完了！不管结果如何——去吃顿好的，睡到自然醒
```

## 五、突发情况应急手册

| 突发情况 | 应对 |
|---------|------|
| **忘带准考证** | 找考点工作人员→去考点办公室→登录研招网下载打印。考点通常有备用打印条件 |
| **忘带身份证** | 找考点工作人员→考点通常有身份核验通道→或去最近的派出所开临时身份证明 |
| **迟到 < 15 分钟** | 通常可以入场。保持冷静，不要因为迟到打乱节奏 |
| **迟到 > 15 分钟** | 该科无法参加（= 0 分）。其他科目继续考——一科 0 分 ≠ 全部白费 |
| **身体不适（肚子疼/头晕）** | 举手找监考老师→考场有医护人员。不要硬撑——安全第一 |
| **答题卡涂错位** | 发现后立刻举手找监考老师→申请换答题卡。越快发现越好 |
| **笔突然写不出** | 带 3-5 支笔就是为了这个——直接换一支，不慌 |
| **旁边考生抖腿/噪音** | 举手跟监考老师说——他们会处理。不要自己跟对方起冲突 |
| **试卷印刷不清** | 举手问监考老师——这是你的权利 |
| **大脑突然空白** | 深呼吸 3 次 → 跳过当前题 → 做后面会的题 → 回头再看。空白是暂时的，别慌 |

### 最重要的 3 条应急原则

1. **任何问题找监考老师**——考场有完整的应急预案，你不是第一个遇到问题的人
2. **一科考砸 ≠ 全盘皆输**——考完一科忘一科，把精力留给下一科
3. **身体 > 考试**——头晕/剧烈疼痛/发烧 = 立即报告，健康远比分数重要

## 六、考前心态管理

### 考前最后一周的心理波动

| 感觉 | 真相 |
|------|------|
| "我什么都忘了" | 不是忘了，是焦虑让大脑宕机。知识都在——考场上看到题目就会激活。这叫「检索性遗忘」，极其普遍 |
| "还有好多没复习" | 永远有没复习完的。考研不是考满分——做对你会的题就够了 |
| "模考很差我肯定不行" | 模拟题 ≠ 真题。张宇 8 套卷 90 分可能对应真题 110+ |
| "我肯定考不上" | 每个人考前都这么想。这是大脑在尝试通过降低预期来保护你——不要理它 |

### 考前 1 天晚上可能睡不着

非常正常。应对：
- 23:00 前上床，即使睡不着也躺着闭眼
- 不碰手机（蓝光 + 焦虑信息 = 双重失眠）
- 如果实在睡不着 → 闭眼听政治带背音频（反而可能睡着）
- 一晚没睡好 ≠ 考试一定会考砸——人体在应激状态下能正常运转
